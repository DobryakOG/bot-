# Папка для изображений

Поместите сюда изображения, которые будет использовать бот.

## Необходимые файлы:

- `banner.png` - Баннер для объявления о заявках (рекомендуемый размер: 800x400px)
- `portfolio.png` - Изображение для системы портфолио (используется в двух местах: в канале создания портфолио и в приветственном сообщении)

## Поддерживаемые форматы:

- PNG (рекомендуется)
- JPEG / JPG
- GIF

## Настройка:

Вы можете изменить название файла баннера в `.env` файле:
```env
IMAGE_BANNER=banner.png
IMAGE_PORTFOLIO=portfolio.png
```

## Примечание:

Если файл баннера не будет найден, бот просто отправит сообщение без изображения.
Все изображения должны находиться в папке `images/`.
